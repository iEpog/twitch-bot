# twitch-bot
Simple Twitch moderation/token bot with Discord Webhook Logger and Command Handler

<h1>Features</h1>
⦁ Moderation Commands
 </br>
 ⦁ Log actions to Discord with Webhooks
 <br>
⦁ Raid/Host Track
 </br>
⦁ Twitch Subscription Tack
 <br>
 ⦁ Simple Command handler
 <br></br>
